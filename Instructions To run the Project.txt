One-time setup (only once per machine):

Open PowerShell in project root:
cd "d:\MSc in AI Resources\Second Semester\Machine Learning\Project\da-claim-assistant"

Create and install Python env:
py -m venv .venv
Activate.ps1
python -m pip install --upgrade pip
pip install -r requirements.txt

Install frontend packages:
cd frontend
npm install
cd ..
